# 入门

<br/>
<br/>

## 快速开始

新建一个文件夹，在该目录下直接使用如下命令
```
git clone https://github.com/MikuBlog/xz-admin.git
```

进入到下载好的项目目录下，使用如下命令进行依赖安装
```
npm install

```

## 目录结构

```
├─api // 存放自行封装的api
│  ├─database
│  ├─dom
│  ├─file
│  ├─http
│  ├─iconfont
│  ├─json
│  ├─message
│  ├─other
│  └─storage
├─assets // 静态资源文件
│  ├─error
│  ├─loading
│  ├─login
│  ├─logo
│  └─welcome
├─components // 第三方组件与自行封装的组件
│  ├─breadcrumb
│  ├─editor
│  ├─icon_select
│  ├─scrollpane
│  ├─SvgIcon
│  ├─tag
│  ├─tree_menu
│  └─tree_table
├─global
├─icons // 存放SVG图标
│  └─svg
├─initial // 第三方插件与过滤器初始文件
├─Layout // 布局组件
├─router // 路由核心文件
├─store // vuex
└─views // 页面布局文件
```
